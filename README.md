# 📊 Vizag Value Mart – Sales Analysis
Interactive dashboard creation - using EXCEL 
> ## 📌 Project Overview

This project analyzes the Vizag Value Mart retail dataset using Microsoft Excel to identify sales patterns, 2 months of customer behavior, and payment trends. The goal of this project is to practice data cleaning, data analysis, and visualization using Excel tools.
The dataset contains retail transaction data including customer information, product categories, sales values, and payment methods.

> ## 📂 Dataset Information
- The dataset includes 4999+ retail transactions with multiple attributes related to customer purchases.
- Bill_No
- Customer_ID
- Gender
- City / Location
- Purchase_ID
- Date
- week
- Time
- Time_Bucket
- Categories 
- Items / Products
- Qty / Units
- Payment_Mode
- Product_Price
- Gross Sales
- Net Sale
- Discount %

> ## 🎯 Project Objectives
- Analyze sales performance across product categories
- Identify customer purchasing behavior
- Evaluate payment method trends
- Discover peak sales time periods
- Build an interactive Excel dashboard

> ## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning
- Conditional Formatting
- Excel Dashboard

> ## 📊 Key Insights
- This project aims to analyze 4999+ sales records with 17 variables, 2 months including customer details (gender and location), purchase information (date, time, and item), product categories, quantities, payment modes, and sales values. By applying data analysis techniques, the project identifies trends in customer buying patterns, popular product categories, peak shopping times, and revenue contributions from different products and payment methods.

- The analysis helps in understanding which products sell the most, which locations contribute more customers, and how different payment modes affect sales performance. Insights from this project can assist store managers in improving inventory planning, marketing strategies, and customer service.

- Overall, this project demonstrates how data analysis can transform retail transaction data into meaningful insights that support better business decision-making and operational efficiency.
